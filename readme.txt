# XML Parser

## INTRODUCTION

### What the program does.
This program require a user input for an XML file. It then parses the file and check for any incorrectness. 
The program uses MyArrayList, MyQueue, MyStack data structures implemented from scratch. 
Testing classes consist of JUnit tests for all the data structures.

## REQUIREMENTS

This program requires JDK1.8 installed on the computer.


## INSTRUCTIONS

### How to run the program

1. Copy the XML file that you want to parse in the data folder.
2. Open CMD and look for where Parser.jar is located.
3. Enter the name of the file copied in data folder with the xml extension. Example:	
	File1.xml
	File2.xml   


Or

1. Using IDE such as Eclipse 
2. Open Project
3. Run project
4. Enter your file or sample files
